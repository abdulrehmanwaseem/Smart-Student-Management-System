# Smart Student Management System

Maine Smart Student Management System ka project complete kar liya hai. Ye ek terminal based python project hai jiska main code `index.ipynb` file ke andar hai.

Extra kaam maine ye kiya ke maine is project ko FastAPI ka use karke `API` k folder mein convert kar k dal diya. Frontend ke liye maine HTML, CSS aur JavaScript ka ek simple template use kiya aur APIs ko usme integrate kar diya.

Project ka code maine deploy bhi kar diya hai, aap is link par ja kar test kar sakte hain:
http://smart-student-management-system.vercel.app

Agar aap is project ko apne system par run karna chahtay hain to bas `start_system.bat` file run kar den terminal me, ya manually open karke bhi chala sakte hain.
